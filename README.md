# Natural Language Processing Assignments
---

## Overview

This repository contains four programming assignments for the Natural Language Processing course at National Tsing Hua University (NTHU). Each assignment focuses on different core NLP concepts and techniques.

---

## Assignment 1: Word Analogy

### Description
Practice training and evaluating word embedding models using the Google Word Analogy task. The assignment covers both using pre-trained embeddings and training custom embeddings on Wikipedia corpus.


---

## Assignment 2: Arithmetic as Language

### Description
Train LSTM recurrent neural networks to perform arithmetic operations by treating mathematical expressions as a sequence generation task.

---

## Assignment 3: Multi-output Learning

### Description
Build a multi-output learning model using BERT for the SemEval 2014 Task 1 dataset, handling both regression and classification tasks simultaneously.

---

## Assignment 4: RAG System using LangChain

### Description
Build a Retrieval-Augmented Generation (RAG) system using LangChain and Ollama to answer questions about cat facts. This assignment covers setting up local LLM inference, creating vector stores, and implementing a complete RAG pipeline.
